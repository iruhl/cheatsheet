Setting up Protractor

Make sure java is installed

Open a comand window and type: java -version You should geta response like this: 

java version "1.7.0_45" Java(TM) SE Runtime Environment (build 1.7.0_45-b18) Java HotSpot(TM) Client VM (build 24.45-b08, mixed mode, sharing)
if you do java is installed properly and you can move on to next section.

If you get an answer like this: 

'java' is not recognized as an in operable program or batch file.
java is not properly installed

Java might still be installed butt not added to the path variable. Open Control Panel\Programsif Java(32-bit) exists ther Java is installd otherwise go to java.com/download/and download and install java.

To add Java to the path variable open Control Panel\Programsand open Java(32-bit) go to tab 'Java' click view and copy the Path column. Open Control Panel\System and Security\Systemand click 'Advanced system Setting' click Enviroment Variables button. In the system variables table find the path variable and add the java path to it. OBS only add the path up to the bin library ex: C:\Program Files (x86)\Java\jre7\bin

Make sure NodeJS is installed

Open a comand window and type: java -version You should geta response like this: 

v0.10.22
If you do NodeJS is installed and you can move on to the next section. Otherwise go to nodejs.org and install NodeJS
